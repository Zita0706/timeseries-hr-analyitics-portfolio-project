# timeseries-hr-analyitics-portfolio-project
 Project Name: HR Analytics: Exploring Unemployment Trends in the United States  Description: This GitHub repository contains a Jupyter Notebook documenting an in-depth analysis of unemployment trends in the United States from 2010 to 2019. Leveraging the Unemployment Kaggle Dataset (2010 - 2020) uploaded by Aniruddha Shirahatti, the analysis explores variations in unemployment rates across different demographic groups, including education level, race, and gender.  Executive Summary: The analysis investigates trends and patterns in unemployment rates, highlighting significant differences based on education level, racial groups, and gender. Key findings underscore the importance of education in accessing economic opportunities and reveal disparities among different demographic groups.  Key Features:  Detailed analysis of unemployment trends from 2010 to 2019 Examination of unemployment rates across education levels, racial groups, and gender Identification of disparities and patterns in unemployment rates Insights into the influence of educational attainment on unemployment rates Exploration of seasonal patterns in unemployment rates Forecasting the men's unemployment rate for the near future, including April 2021 Tools Used:  Python libraries: pandas, numpy, matplotlib, seaborn, statsmodels, pingouin Jupyter Notebook for code execution and documentation Dataset Source: Unemployment Kaggle Dataset (2010 - 2020) by Aniruddha Shirahatti  Instructions:  Clone the repository to your local machine. Open the Jupyter Notebook file to explore the analysis and findings. Run the code cells to reproduce the analysis or modify as needed.
